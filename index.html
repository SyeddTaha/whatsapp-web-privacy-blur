What gets blurred

Seven independent
blur targets

Toggle each category separately. Blur only what matters for your situation.

💬
Chat Messages
Every bubble in the open conversation — incoming and outgoing. Hover the bubble to read it.
📋
Chat Previews
The last message snippet shown under each contact name in the sidebar. Hidden until you hover the row.
🖼️
Media Thumbnails
Images, videos, stickers, audio players, and document previews inside chat. Toggled separately from text.
🗂️
Media Gallery
The full-screen lightbox viewer. Opens blurred whenever you click a photo to expand it.
⌨️
Message Input
What you're typing in the compose box is blurred at rest. Auto-reveals when you click into it or hover.
👤
Names
Contact and group names in the sidebar, the chat header, and inside group messages.
🧑‍🤝‍🧑
Profile Pictures
Avatars everywhere — sidebar, chat header, and message threads. Blur covers the full circle.
Reveal behaviour

Three ways to
unblur

Choose how content reveals when you need to see it.

Default
Hover to reveal
Move your mouse over any blurred element to see it. Move away and it blurs again. Everything else on the page stays hidden.
Option
Instant reveal
Turns off the 180ms fade transition. The unblur is immediate on hover — no animation, snappier feel if the transition bothers you.
Option
Hover app to reveal all
When this is on, moving your mouse anywhere over the WhatsApp Web tab unblurs everything at once. Useful when you want to read normally for a stretch without hovering item by item. Move your mouse off the browser window and everything blurs again.
Installation

Up in about
60 seconds

No Chrome Web Store account needed. Load it directly as an unpacked extension.

1
Download the ZIP
Click the download button above or grab it directly from GitHub. Unzip it anywhere — your Downloads folder is fine.
2
Open Extensions
In Chrome, Edge, or Brave go to chrome://extensions in the address bar. In Edge it's edge://extensions.
3
Enable Developer Mode
Toggle the Developer mode switch in the top-right corner of the extensions page. This lets you load extensions from local files.
4
Load Unpacked
Click Load unpacked, then select the unzipped folder (whatsapp-web-privacy-blur-main/). Don't select the ZIP — select the folder inside it.
5
Open WhatsApp Web
Go to web.whatsapp.com. Blur is active immediately. If you had it open already, do a hard refresh (Ctrl+Shift+R / Cmd+Shift+R).
6
Configure from the popup
Click the extension icon in your toolbar to open the control panel. Toggle individual blur categories and reveal behaviour to match your needs. Settings save automatically.
Under the hood

How it actually
works

No magic. Just CSS, a MutationObserver, and one insight that makes message blur work correctly.

Message blur
Blur the row, not the span
WhatsApp's parent divs use overflow:hidden. Blurring inner text spans clips the glow at the container edge — nothing looks blurred. The fix: blur the whole .message-in / .message-out row. These class names are stable and not obfuscated.
CSS injection
One <style> tag, rebuilt on change
A single <style> tag is injected into WhatsApp's <head> and regenerated whenever settings change. No per-element style attribute pollution. A second MutationObserver watches <head> and re-injects it if WhatsApp removes it.
Avatar detection
Structural scan, not URL matching
Avatars share blob URLs with media, so img[src^="blob:"] would catch everything. Instead a debounced MutationObserver scans for small square images (28–82px) and tags them with .wpb-av. CSS targets that class.
Selector strategy
data-testid over class names
WhatsApp rotates its CSS class names with every deploy. data-testid attributes stay stable — they use them for their own E2E tests. Names use span[title] anchors which have been reliable for years.
🔒

Completely private by design

The extension never reads your message content. It only applies CSS filters to DOM elements. No data leaves your browser. No analytics, no tracking, no network requests. All settings live in chrome.storage.sync — local to your browser, optionally synced across your own Chrome profile.
